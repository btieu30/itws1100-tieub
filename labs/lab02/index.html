Lab 2: HTML & CSS

The goal of this lab was to build our resumes in HTML and CSS. We were given multiple resume templates to build on. We were also supposed to push our changes to the GitHub repository, reflect the changes on the VM, and make sure the changes were live.

Approach

I completed this lab by using one of the resume templates and filling it in with my own information. I used the HTML and CSS presentations from class to help me with the structure and styling of the page. I made sure the webpage looked how I wanted it to locally before pushing to GitHub and pulling the changes on the VM.
