Lab 1: VM, GitHub & Azure Setup

The goal of this lab was to set up our virtual machines on Azure, create a GitHub account (if we didn't already have one), create a repository to store the coursework for the class, and deploy a simple webpage to the VM. This involved following a number of tutorials provided to us in the lab documents.

Approach
